# Feishu VC

Use for video conference records and meeting metadata. For Minutes transcript/content, use `feishu-minutes` or `feishu-note` when appropriate.

## Read operations

Searching meetings, reading meeting detail, attendees, records, and meeting events is safe when the user has access.

## CLI discovery

```bash
lark-cli vc --help
```
