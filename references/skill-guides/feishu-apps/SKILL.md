# Feishu Apps

Use for Feishu app development and deployment workflows.

## Reads and dry runs

Inspecting app metadata, local build output, and deployment status is safe.

## Writes

Creating, publishing, deploying, or changing app access scope affects users. Show a preview and ask for confirmation.

## CLI discovery

```bash
lark-cli apps --help
```
