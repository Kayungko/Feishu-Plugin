# Feishu Minutes

Read `../feishu-shared/SKILL.md` first.

## Safe operations

- Search minutes by keyword, owner, participant, or time range.
- Fetch AI summary, chapters, action items, or transcript.
- Generate a doc summary from minutes.

## Confirm first

- Download audio/video recordings.
- Share minutes content to a group.
- Create or assign tasks to other people from action items.

## Common workflow

For "把会议纪要整理一下":

1. Fetch minutes summary and transcript if needed.
2. Extract decisions, action items, owners, due dates, risks.
3. Create a Feishu doc or task draft.
4. Ask before assigning tasks or sending messages.
