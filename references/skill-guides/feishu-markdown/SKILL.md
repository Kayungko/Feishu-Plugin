# Feishu Markdown

Use for Drive-native Markdown files, not for converting Markdown into Docx. For Docx creation from Markdown, use `feishu-doc` or `feishu-workflows`.

## Read operations

Fetching Markdown content and comparing text is safe.

## Writes

Creating or overwriting Markdown files changes Drive content. Show a preview before writing.

## CLI discovery

```powershell
lark-cli markdown --help
```
